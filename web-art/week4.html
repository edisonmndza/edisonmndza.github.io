Aurora Cannabis Inc. [ACB]

The idea of this project is to use the market value of this specific stock (link below) and have an animated marijuana plant that will either be:
1. Barely grown - meaning that the stock value is down compared to yesterday
2. Mid grown - meaning that the stock value is on par (within a certain percent range) compared to yesterday
3. Fully grown - meaning that the stock value is up compared to yesterday.

The stock market value will be used to determine which state the plant should be in.
Link to the target website: ACB Aurora Cannabis
